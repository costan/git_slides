h1. Committing Explained

# Stage all changes except for deletion 
** @git add .@

# Show me what's about to happen
** @git status@

# Stage deletions
** @git add -u .@

# Make the commit.
** @git commit -m "Initial homepage."

h1. Faster Committing.

# Stage *all* changes 
** @git add -A .@

# Show me what's about to happen
** @git status@

# Make the commit.
** @git commit -m "Initial homepage."

h1. Fastest Committing.

# Stage *all* changes, make the commit.
** @git commit -a -m "Initial homepage."

h1. Thinking.
